# Simple_Sales_Dashboard

Objective

Build an interactive dashboard to analyze sales performance by Region, Category, and Month.

Tools Used

- Power BI
- CSV Dataset

Dashboard Components

1. Sales Trend by Month (Line Chart)
2. Sales by Region (Bar Chart)
3. Sales by Category (Donut Chart)
4. Region and Category Slicers

Key Insights

1. Technology category generated the highest sales.
2. West region contributed the highest revenue.
3. Sales showed consistent growth from January to June.
4. Furniture and Technology together accounted for most of the revenue.

Files Included

- Sales_Data.csv
- Dashboard_Screenshot.png
- Insights.txt
- README.md

Outcome

Created an interactive sales dashboard to visualize business performance and identify top-performing regions and categories.
